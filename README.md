# Shopalot

E-commerce web app made for a frontend developer course assignment.

### Stack

- Vite
- React
- TypeScript
- Styled Components

### Packages

- React Router
- Zustand state manager
- Styled Components
- Framer Motion
- React Hook Form
- Yup

### Project setup

- Clone or fork the repository
- Run `npm install` in the terminal.
- `npm run dev` to start.
